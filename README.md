# Predicting-London-House-Prices-with-Multiple-Linear-Regression
This project uses multiple linear regression to predict London house prices using property size, location, tenure, property type, and energy rating.

The model uses:
- log-transformed price as the target
- log-transformed floor area
- grouped outcodes and property types
- a preprocessing pipeline with imputation, scaling, and one-hot encoding
- baseline comparison and model improvement experiments
- an interactive prediction dashboard

The raw dataset is not included in this repository. It should be downloaded separately from Kaggle
